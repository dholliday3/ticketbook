It's likely that we'll have many tickets in a single PR or commit. Being able to link them would be nice, but it might be cumbersome. It needs to be automatic, or else it won't work. Automatic might mean deterministically automatic with metadata and/or some convention that LLMs know to follow to link tickets to a given commit and/or PR that when followed works properly.
